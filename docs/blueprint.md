# **App Name**: GanhaFlow Invest

## Core Features:

- User Authentication: Secure user login and registration with Gmail and password options.
- Dashboard: Display user's available balance and account status.
- Investment Machines: List machines (Bronze, Silver, Gold, Platinum, Diamond) with their respective investment amounts and return values.
- Investment Cycle Simulation: Simulate investment cycles. Calculate when each investment cycle will yield returns, using the 7-day return cycle.
- Deposit System: Enable users to deposit funds via PIX with predefined amounts (R$10, R$20, R$50, R$100, R$200).
- Menu Navigation: Provide navigation between 'Machines', 'My Machines', and 'Withdrawal' sections.
- Transaction Analysis: Suggest optimal investment strategies and risk assessment using deposit and withdrawal transaction histories.

## Style Guidelines:

- Background color: Dark graphite (#1E1E1E) to create a Dark Mode theme.
- Primary color: Neon green (#39FF14) for main actions and success indicators, providing a vibrant and energetic feel.
- Accent color: Yellow-orange (#FFC300) to highlight important elements and CTAs, ensuring key information stands out.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Use simple, minimalist icons to represent various actions and categories, ensuring clarity and ease of use.
- Employ a card-based layout with rounded borders to present information in a structured and visually appealing manner.
- Implement subtle glow effects on buttons and borders to add interactivity and a sense of dynamism.
